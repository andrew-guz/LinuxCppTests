## Containers, their internal structure description and main operations complexity.

### Stack

FILO


### Queue

FIFO


### Vector

Based on array with additional size so will not relocate always.

| Operation | Complexity |
| - | - |
| add | O(1) |
| insert | O(n) |
| get | O(1) |
| remove | O(n) |


### deque

Array chunks combined in a list

| Operation | Complexity |
| - | - |
| add | O(1) |
| insert | O(n) |
| get | O(1) |
| find | O(n) |
| remove | O(n) |
